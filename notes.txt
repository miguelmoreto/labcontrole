

To generate a executable:


Creating a distribution enviroment (Linux):
    conda create --name labcontrole python nomkl --no-default-packages
    conda activate distenv
    conda install numpy
Creating a distribution enviroment (Windows):
    conda create --name labcontrole python --no-default-packages
    conda install conda-forge::blas=*=openblas
    conda install -c conda-forge numpy
Activate the enviroment:
    conda activate labcontrole
Install packages:
    conda install scipy
    conda install matplotlib
    conda install -c conda-forge control slycot
    pip install --upgrade PyQt5-sip
    conda install -c anaconda pyqt
    conda install -c conda-forge libstdcxx-ng
    conda install -c conda-forge pyinstaller
Build the binary (Linux):
    pyinstaller --noconfirm --clean --onedir --add-data="MainWindow.ui:." --add-data="HelpWindow.ui:." --add-data="matplotlibwidget.py:." --add-data="./help/*:./help" --exclude-module=sqlalchemy --hidden-import=PyQt5.sip LabControl3.py
    For pyInstaller 6:
        pyinstaller --noconfirm --clean --onedir --contents-directory "."  --add-data="MainWindow.ui:." --add-data="HelpWindow.ui:." --add-data="matplotlibwidget.py:." --add-data="./help/*:./help" --exclude-module=sqlalchemy --hidden-import=PyQt5.sip --hidden-import=numpy.core.multiarray LabControl3.py
Build the binary (windows):
    pyinstaller --noconfirm --clean --onedir --add-data="MainWindow.ui;." --add-data="HelpWindow.ui;." --add-data="matplotlibwidget.py;." --add-data="./help/*;./help" --exclude-module=sqlalchemy --hidden-import=PyQt5.sip LabControl3.py
    For pyInstaller 6:
        pyinstaller --noconfirm --clean --onedir --contents-directory "." --add-data="MainWindow.ui;." --add-data="HelpWindow.ui;." --add-data="matplotlibwidget.py;." --add-data="./help/*;./help" --exclude-module=sqlalchemy --hidden-import=PyQt5.sip --hidden-import=numpy.core.multiarray LabControl3.py

conda deactivate

Utility commands:
pyrcc5 images.qrc -o images_rc.py

# Resize to specific width:
convert input1.png -geometry 480x input1.png

Problems with MESA-LOADER in Anaconda:
$ cd /home/$USER/anaconda3/envs/$ENV/lib
$ mkdir backup  # Create a new folder to keep the original libstdc++
$ mv libstd* backup  # Put all libstdc++ files into the folder, including soft links
$ cp /usr/lib/x86_64-linux-gnu/libstdc++.so.6  ./ # Copy the c++ dynamic link library of the system here
$ ln -s libstdc++.so.6 libstdc++.so
$ ln -s libstdc++.so.6 libstdc++.so.6.0.19

TO COMPILE SLYCOT (testado no Linux Mint 21.2):

pip install scikit-build
sudo apt-get install cmake
sudo apt-get install gfortran
sudo apt-get install libopenblas-dev
export FC=/usr/bin/gfortran

git clone --recurse-submodules https://github.com/python-control/Slycot.git
cd Slycot
pip install -v .

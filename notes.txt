

To generate a executable:

1) Activate conda in the terminal.

2) conda install -c conda-forge pyinstaller
3) pip install --upgrade PyQt5-sip
4) pyinstaller --clean --exclude-module sqlalchemy --hidden-import PyQt5.sip LabControl3.py

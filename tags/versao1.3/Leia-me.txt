Instruções para geração do executável:


Não sei por que, mas para o py2exe incluir o módulo signal é necessário
que nos programas seja colocado:

from scipy import signal

pois somente "from scipy import *"

não funciona. Então, é necessário modificar isso no módulo controls:

C:\Python25\Lib\site-packages\controls\__init__.py

substituindo acrescentando a linha "from scipy import signal".
